1. 重写了大部分界面元素
   特定形状的砖块总是使用固定的颜色，玩家可以看颜色区分砖块形状。
   ; 修改了生成砖块的代码，同时生成颜色，颜色和砖块形状一起保存
   游戏暂停和结束添加了相应的界面显示
   ; 暂停时显示一个文本标签覆盖游戏
   ; 游戏结束时在已有砖块的位置上涂上深灰色
2. 修改了原版翻转的一个问题，现在如果直接翻转会撞上其他方块，
     但是翻转同时平移一格可以不和其他方块重叠，同样可以进行翻转。
     （注意，长条在边上翻转需要平移两格的仍然不允许翻转）
   ; 在翻转失败时尝试左移一格或者右移一格。
3. 增加等级和消除行数，速度由等级决定，修改了记分系统
   ; 在砖块落到底部时根据高度、当前等级计算分数
   ; 消去行时计算消除行数，且一次消除的越多得分越多
   ; 150以内每30行加1级，以后每50行加1级，最多不超过17
4. 开始游戏可以选择级别，可选择的范围从0～9
   ; 初始化时添加一个级别的参数
5. 重定义了上下键，不使用上下作为翻转，下作为快下，ZX键或JK键作为翻转键；
   同时支持ASD键作为方向键使用，PageUp、PageDown作为旋转
   同时支持对应Dvorak键盘映射时对应位置的按键（AOE;QHT）
   重开游戏放到了F2上，退出放到了Esc上
6. 改变了S型Z型砖块旋转的形状，两次旋转应保持形状而不该发生平移
   ; 修改了定义砖块的常量数组
7. 显示将要产生的下一个砖块的形状
   ; 添加一个新的画图区域
   ; 并修改描画砖块的代码使之可以作用在不同画图区域上
   ; 每次产生新的砖块时重绘，显示下一个砖块的形状
